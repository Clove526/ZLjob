## 附录：拍摄/制作建议

### 录制工具
- **屏幕录制**: OBS Studio / Screen Studio（推荐，有鼠标高亮）
- **后期剪辑**: 剪映专业版 / Premiere Pro

### 配乐建议
- 开场：悬疑紧张风格
- 剧情引入：叙事感轻音乐（如RPG游戏BGM）
- 核心玩法：节奏感较强的电子乐
- 价值总结：温暖鼓舞风格

### 字幕风格
- 关键词/数据：金色或蓝色高亮字幕
- 技术说明：底部白色半透明字幕
- 策略名称：对应颜色的图标+文字

### 时长控制技巧
- 如果时间不够，可缩减"剧情引入"部分（Part 3）至20秒
- 核心玩法（Part 4）是最重要的展示段落，不要压缩
- 创新亮点（Part 5）可根据赛道侧重点选择性展示：
  - AI+求职赛道：重点展示Part 4 + 亮点1
  - AI+创意赛道：重点展示亮点2 + 亮点3
